# Source Code

This folder contains runnable tools and experiments.

## Current tools

- `v4l2_capture_demo.c` - a minimal V4L2 capture program for Linux camera pipeline validation
- `camera_log_analyzer.c` - a simple log classifier for common camera bring-up failures
- `sensor_reg_tool.c` - a register table parser and diff tool for sensor tuning
- `media_graph_dump.c` - a sysfs-based helper that lists video nodes and related device links
